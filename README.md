# Queen's Computing Course Planner

Welcome to the GitHub repository of Course Planner @ Queen's! We are a student-run site that now proudly supports over 1500 students at Queen's!

The deployed link (via Vercel) is at [https://courseplanner.ca](https://courseplanner.ca)

To start the development server: enter `npm run dev`

The first time you execute your file, please make sure to start with `npm install` to download any packages used.
